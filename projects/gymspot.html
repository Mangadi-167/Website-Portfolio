Website GYM Spot

Improve User Experience by easily finding gym recommendations, and helping promote gym owners

My Role in This Project

The Gymspot management app project was designed to streamline gym operations, from member registration to trainer engagement, and help gym owners reach more customers. In this development, I played the following roles:

  • UI/UX Designer:

    Design an intuitive user flow and engaging interface, ensuring a smooth experience for both users and gym owners.

  • Frontend Developer:

    Translate visual designs into interactive and responsive web code, using expertise in Tailwind CSS for modern layouts and JavaScript for dynamic functionality.

  • Backend Developer:

    Building a robust server-side architecture using PHP and the Laravel framework, managing a MySQL database for member data, schedules, and transactions, and ensuring application security and scalability.

The main focus of this project is to help users find fitness centers based on where the user lives, and then help fitness center owners promote their fitness centers so they can reach more consumers.
