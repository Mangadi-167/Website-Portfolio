AI Chatbot-Based Tanah Lot Site

Improve User Experience with Modern Look, Real-Time Customer Service using AI Chatbot, and Online Ticket Purchase

My Role in This Project

The Tanah Lot website project was a comprehensive effort to modernize the digital experience. I played a key role in this project as:

  • UI/UX Designer:

    Responsible for designing intuitive user interfaces and smooth user experiences, ensuring easy navigation and appealing aesthetics.

  • Frontend Developer:

    Implementing designs into responsive and interactive visual code using HTML, CSS (specifically TailwindCSS), and JavaScript.

  • Backend Developer:

    online ticket purchasing functionality, managing dashboards, and ensuring optimal application performance with Python and the Django framework, as well as managing MySQL databases.

Developed a complete (end-to-end) dynamic web platform for Tanah Lot, including all backend and frontend logic, as well as integration with third-party services. The project was built using Python with the Django framework, and its responsive user interface was built with Tailwind CSS and JavaScript.
